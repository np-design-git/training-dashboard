## HOW TO USE THIS LOG

**One entry per session. Fill in immediately after training while it's fresh.**
**Copy the relevant template block below. Delete unused fields (don't leave them blank).**
**Date format: DD.MM.YYYY — always.**

**What the coach needs from every entry:**
- Date + session type
- Oura score that morning (tells me your recovery state)
- Actual sets × reps × weight for every working set
- Any flags: form breakdown, pain, fatigue, hip/scar/knee feedback
- RPE (1–10)

**What you do NOT need to write:**
- Warm-up details (assumed unless something notable happened)
- "Exercise was controlled" — only flag if it WASN'T
- Phase context / post-op week (I track this from the profile)
